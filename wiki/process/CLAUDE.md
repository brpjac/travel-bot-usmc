# Process

Getting orders requested, funded, and travel booked — on time. If you only read
one thing here, read [t3-timelines.md](t3-timelines.md): late submissions are
the #1 failure mode.

## T3 quick table (deadlines before departure day D)

| What | When | Claim |
|---|---|---|
| Group charter T3 → final approver | D-65 | [C020] |
| OCONUS commercial air T3 → final approver | D-50 | [C022] |
| CONUS commercial air/bus T3 → T3 Manager | D-35 | [C024] |
| GTCC activated, orders attached | D-10 | [C026] |
| Full ladder + DMO deadlines | [t3-timelines.md](t3-timelines.md) | |
