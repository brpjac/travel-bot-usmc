# Contacts and Systems

## MIU contacts

| Who | For |
|---|---|
| **MIU_Operations@usmc.mil** | HTC drill travel notification, NLT 1 week prior [C016] |
| **S-1** | Orders questions, currency issues, anything this wiki can't settle |
| **S-4** | Billeting Request Form for HTC lodging [C016] |
| **DET leadership** | Courtesy drills (RTAMMS), Appropriate Duty (NAVMC 799) |

## Systems map

| System | What it does | You touch it when |
|---|---|---|
| **Correspondence Tracker** | MIU orders request intake | Requesting Offsite IDT / AT / ADOS orders |
| **DTS** | Defense Travel System — booking + vouchers | Active Reserve/AD travel; events where DTS is authorized |
| **DTS-R** | DTS for Reserves | The $750 HTC IDT travel voucher |
| **T3 database** | MARFORRES transportation requests (TEEP events) | Commercial transport outside DTS (ForO 3000-52.1, Ch 6) |
| **MROWS** | Marine Reserve Order Writing Service | Where reserve orders get written (ForO 3000-52.1, Ch 7) |
| **RTAMMS** | Reserve Training and Administration Management System | Courtesy drills |
| **APACS** | OCONUS clearance | Any OCONUS travel |
| **Drill Manager / MIU Drill Submission Page** | Drill administration | Appropriate Duty (NAVMC 799) |

(MIU Orders-Type Matrix, 6 Mar 2026; ForO 3000-52.1, Ch 6-7)

## Beyond the unit

- **MARFORRES/MSC T3 Manager** — the final approver for T3 requests
  ([../process/t3-timelines.md](../process/t3-timelines.md))
- **DMO** (Distribution Management Office) — receives TOP/TOT requests and does
  the actual ticketing
- **MSC G-3/5 and MSC Chief of Staff** — the late-submission endorsement chain
  ([../process/t3-waivers.md](../process/t3-waivers.md))
