# Orders Types

What each orders type is, when it applies, and what it unlocks. Start with
[choosing-orders-type.md](choosing-orders-type.md) if you don't know which one
you need.

## Quick table (SMCR Marines)

| Your trip | Orders type | Page |
|---|---|---|
| Regular drill at your HTC | IDT | [idt.md](idt.md) |
| Drill >50 mi from HTC | Offsite IDT | [offsite-idt.md](offsite-idt.md) |
| Training >7 days | AT | [at.md](at.md) |
| Duty <7 days needing entitlements | ADOS | [ados.md](ados.md) |
| Activation/mobilization | Mobilization | [mobilization.md](mobilization.md) |

Active Reserve and Active Duty Marines: all travel goes through DTS — see
[../process/dts-and-claims.md](../process/dts-and-claims.md).
