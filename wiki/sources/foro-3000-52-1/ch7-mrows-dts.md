# ForO 3000-52.1, Chapter 7 — MROWS and DTS (verbatim)

## 1. Introduction (p. 7-1)

T3 Event funding is divided into numerous categories and combinations
depending on the type of exercise/event being supported. Due to these
complexities it is critical that orders be written correctly with the proper
funding data and T3 information. Valuable training time and money can be lost
when orders are not written correctly. This chapter is to assist order writers
in preparing TEEP funded orders in MROWS or DTS.

## 2. Marine Reserve Order Writing Service (MROWS) (pp. 7-1 to 7-2)

The Marine Corps' system of record used to create orders for reserve
personnel.

a. **TEEP Orders.** Orders written in support of a TEEP funded
exercises/event.

(1) Orders must contain the TEEP Event Number; a 7 place alpha numeric
identifier specific to a TEEP exercise/event, i.e. AFRICAN LION TEEP event
number = F23-7870. This information can be obtained from the T3 database or
the operations section of the units command.

(2) Orders must contain the SIC; a 3 place alpha numeric identifier specific
to a TEEP funded exercise/event, i.e. AFRICAN LION SIC = DZ0. This information
can be obtained from the T3 database or the operations section of the units
command.

(3) TEEP orders use only the following PIDs:
(a) PID 1 — AT Travel and Per Diem. Funds reservists while on AT.
(b) PID 2 — IDT. Funds reservists travel and per diem while on offsite drill.
(c) PID 3 — ADOS. Funds reservists pay and allowances, travel and per diem
while performing additional duty other than AT.
(d) PID SIF. Funds reservist travel and per diem when in support of a CJCS
sponsored conference. In MROWS on the travel and per diem blocks, select
Command Code as "Force" and PID as "SIF". Use PID 3 for pay and allowances.
Cannot be used to support Navy personnel; must use TEEP TAD.
(e) PID CTP/CTA. Funds reservist travel (airline ticket only) when in support
of a CJCS sponsored exercise. In MROWS in the travel block, select Command
Code as "Force" and PID as "CTP" if reservist is on AT. Select PID as "CTA" if
reservist is on ADOS. Does not fund per diem. Use PID 1 for AT or PID 3 for
ADOS. Does not fund travel for CJCS conferences. Cannot be used to support
Navy personnel; must use TEEP TAD.
(f) PID IRT ADOS. Funds reservist pay and allowances, travel and per diem
while in support of an IRT project. In MROWS select the Command Code as
"Force" and PID as "IRT".

b. MROWS orders in support of a TEEP and Non-TEEP funded events must be
submitted and reach "Pending Fund Approval" status before the TEEP request is
submitted by the unit. The travel manifest must include the MROWS tracking
number (enter only the 7 digit tracking number) before submitting the TEEP
request. If tracking numbers are not included, TEEP request will be denied. A
copy of authenticated orders must be attached to the TOP request prior to
ticketing. MROWS orders for Non-TEEP funded events that are DTS eligible do
not require a T3 request.

c. **Liquidation of orders.** DD Form 1351-2 (travel claim) or DTS voucher
must be submitted no later than 5 working days upon completion of travel by
the individual traveler to their administrative support section.

## 3. Defense Travel System (DTS) (p. 7-2)

Primary system used to create orders on active duty/active reserve personnel
while on TAD and reservists executing DTS supported MROWS order-types.

a. **TEEP Funded Orders.** Orders written in support of a TEEP
exercise/event.

(1) TEEP TAD. Funds Active Duty/Active Reserve, Active Duty Navy and Reserve
Navy. Funds are requested by submitting a TEEP request in T3, once request is
approved, a Financial Information Pointer (FIP) is created and used to create
an LOA in DTS.

(2) SIF. Funds AD/AR personnel and DTS eligible reservists on MROWS orders in
support of a CJCS sponsored exercises/conference. DTS cross org code will be
established by supported MARFOR and traveler will select the proper code to
fund travel and per diem. Traveler must submit name, rank, EDIPI and travel
cost estimate to MARFORRES G-3/5 action officer in charge of the CJCS
exercise/event.

(3) IRT TAD. Funds active duty/active reserve personnel travel and per diem in
support of an IRT project. Funds are requested by submitting a TEEP request in
T3, once request is approved, a Financial Information Pointer (FIP) is created
and used to create an LOA in DTS.

b. **Liquidation of orders.** DTS Travel vouchers must be submitted within 5
working days after completion of travel by the individual traveler.
